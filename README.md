# Mini Catalog App

Mini Catalog App, Flutter kullanılarak geliştirilmiş basit bir mobil katalog uygulamasıdır.  
Bu proje eğitim amaçlı geliştirilmiştir.

---

## Proje Özellikleri

- Ürün listeleme
- Ürün detay sayfası
- Ürün arama
- Sepete ürün ekleme
- Sepetten ürün silme
- GridView tabanlı ürün kartları
- Sayfalar arası geçiş (Navigator)
- Responsive ve modern kullanıcı arayüzü

---

## Kullanılan Teknolojiler

- Flutter
- Dart
- Material Design

---

## Projeyi Çalıştırma

### 1. Projeyi klonlayın

```bash
git clone GITHUB_LINK
```

### 2. Proje klasörüne girin

```bash
cd mini_catalog_app
```

### 3. Gerekli paketleri yükleyin

```bash
flutter pub get
```

### 4. Uygulamayı çalıştırın

```bash
flutter run
```
